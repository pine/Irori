# Irori

REST x RSS based notification system

## Dependencies

- Node.js v4.2.1
- MongoDB
- [OpenShift](https://www.openshift.com/)
- [koa](http://koajs.com/)
- [fly](https://github.com/bucaran/fly)

## Getting started

```
$ npm install --production
$ npm start
```
